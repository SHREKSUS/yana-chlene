# Изображения обложек книг

Поместите изображения обложек книг в эту папку.

## Формат файлов
- Формат: PNG, JPG или JPEG
- Рекомендуемый размер: 200x300 пикселей или пропорционально
- Имя файла: используйте понятные имена, например: `algebra_10.png`, `war_and_peace.jpg`

## Использование
В базе данных укажите путь к изображению в формате:
- `assets/images/название_файла.png`

## Placeholder
Если изображение не указано или не найдено, будет использоваться иконка книги по умолчанию.
